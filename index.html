Panduan Penggunaan Kalkulator
Angka dan Operasi:
  • Klik tombol angka dan operasi untuk memasukkan perhitungan.
  • Perkalian: Gunakan simbol x pada kalkulator atau tanda bintang (x) di keyboard.
  • Pembagian: Gunakan simbol รท pada kalkulator atau tanda garis miring (/) di keyboard.
  • Hapus Angka: Gunakan tombol untuk menghapus angka terakhir.
  • Bersihkan: Klik tombol "C" untuk menghapus semua angka di layar.
  • Hitung: Klik "=" untuk mendapatkan hasil perhitungan.
Keyboard:
  • Gunakan tombol angka dan operator pada keyboard untuk mengetik.
  • Tekan Enter atau "=" untuk menghitung.
  • Gunakan x untuk perkalian dan / untuk pembagian.
  • Tekan Backspace untuk menghapus angka terakhir.
  • Tekan Esc untuk membersihkan layar.
